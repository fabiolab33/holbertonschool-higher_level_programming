# Python Data Structures

This repository contains solutions to **Python tasks** focused on **lists, tuples, and strings**. 

---

## Project Description

The goal of this project is to become familiar with **lists, tuples, and string manipulation** in Python. Each task focuses on a different aspect of data structures, including:

- Iterating and printing lists
- Accessing elements safely
- Modifying lists
- Working with tuples
- Conditional operations on lists
- Handling edge cases (empty lists, negative indices, out-of-range indices, etc.)

---

## Tasks

1. **Print a list of integers**  
   `print_list_integer(my_list)` — Prints all integers of a list using `str.format()`.  

2. **Secure access to an element in a list**  
   `element_at(my_list, idx)` — Returns an element at a specific index safely.  

3. **Print a list in reverse**  
   `print_reversed_list_integer(my_list)` — Prints integers of a list in reverse order.  

4. **Replace an element in a copy**  
   `new_in_list(my_list, idx, element)` — Returns a copy of the list with the element replaced.  

5. **Remove 'c' and 'C' from a string**  
   `no_c(my_string)` — Returns a new string with all characters `c` and `C` removed.  

6. **Print a matrix of integers**  
   `print_matrix_integer(matrix)` — Prints a matrix row by row using `str.format()`.  

7. **Tuples addition**  
   `add_tuple(tuple_a, tuple_b)` — Returns a tuple with the sum of the first two elements of each tuple.  

8. **Multiple returns**  
   `multiple_returns(sentence)` — Returns a tuple with the length of a string and its first character.  

9. **Find the max integer**  
   `max_integer(my_list)` — Returns the biggest integer in a list.  

10. **Multiples of 2**  
    `divisible_by_2(my_list)` — Returns a list of booleans indicating whether each element is divisible by 2.  

11. **Delete at index**  
    `delete_at(my_list, idx)` — Deletes an element at a specific index in a list without using `pop()`.  

12. **Switch values**  
    Swaps the values of two variables and prints them in exactly 5 lines.
