# JavaScript Warm Up

This project is part of the higher level programming curriculum. It introduces the basics of JavaScript using Node.js by solving a series of scripting tasks.

## 📚 Project Description

In this project, we learn how to:
- Run JavaScript scripts using Node.js
- Work with variables (`var`, `let`, `const`)
- Use control flow structures (`if`, loops)
- Work with functions
- Handle command line arguments (`process.argv`)
- Work with arrays and objects
- Use recursion
- Create and export modules

All scripts are executed on Ubuntu 20.04 LTS using Node.js (version 14.x) and must follow semistandard style guidelines.

---

## 📁 Files Description

| File | Description |
|------|------------|
| `0-javascript_is_amazing.js` | Prints "JavaScript is amazing" using a constant variable |
| `1-multi_languages.js` | Prints 3 lines using multiple `console.log` statements |
| `2-arguments.js` | Prints a message depending on the number of arguments |
| `3-value_argument.js` | Prints the first argument passed to the script |
| `4-concat.js` | Prints two arguments in the format "arg1 is arg2" |
| `5-to_integer.js` | Converts and prints the first argument as an integer |
| `6-multi_languages_loop.js` | Prints 3 lines using an array and a loop |
| `7-multi_c.js` | Prints "C is fun" x times based on an argument |
| `8-square.js` | Prints a square of size defined by an argument |
| `9-add.js` | Adds two integers using a function |
| `10-factorial.js` | Computes factorial using recursion |
| `11-second_biggest.js` | Finds the second biggest integer from arguments |
| `12-object.js` | Modifies an object property |
| `13-add.js` | Exports a function that adds two integers (module system) |

## ⚙️ Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- All files interpreted on Ubuntu 20.04 LTS using Node.js (v14.x)
- All files must end with a new line
- The first line of all files must be:
  ```bash
  #!/usr/bin/node
  ```

  ---

  ## ▶️ How to Run

  - Make scripts executable:
  ```bash
  chmod +x filename.js
  ```
  - Run script:
  ```bash
  ./filename.js [arguments]
  ```
