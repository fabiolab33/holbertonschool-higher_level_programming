Test Cases: Normal Usage
===========================

Test with integers, divide by 3:
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

    >>> matrix
    [[1, 2, 3], [4, 5, 6]]

Test with floats:
    >>> matrix_divided([[1.5, 2.5], [3.5, 4.5]], 2)
    [[0.75, 1.25], [1.75, 2.25]]

Test with negative numbers:
    >>> matrix_divided([[-6, -9, -12], [-3, -15, -18]], 3)
    [[-2.0, -3.0, -4.0], [-1.0, -5.0, -6.0]]

Test with mixed int and float:
    >>> matrix_divided([[10, 20.5], [30, 40.5]], 2)
    [[5.0, 10.25], [15.0, 20.25]]

Test divide by negative number:
    >>> matrix_divided([[10, 20], [30, 40]], -2)
    [[-5.0, -10.0], [-15.0, -20.0]]

Test single row matrix:
    >>> matrix_divided([[1, 2, 3]], 1)
    [[1.0, 2.0, 3.0]]

Test single element matrix:
    >>> matrix_divided([[5]], 2)
    [[2.5]]

Test with large numbers:
    >>> matrix_divided([[1000, 2000], [3000, 4000]], 100)
    [[10.0, 20.0], [30.0, 40.0]]

Test rounding to 2 decimals:
    >>> matrix_divided([[1, 2], [3, 4]], 3)
    [[0.33, 0.67], [1.0, 1.33]]
